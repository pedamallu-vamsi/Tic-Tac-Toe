# Tic Tac Toe

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Play directly in your browser with a clean UI and smooth gameplay.

## Live Demo

https://tic-tac-toe-seven-zeta-60.vercel.app/

## Features

* Two-player gameplay
* Real-time win detection
* Draw detection
* Responsive design
* Clean UI

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run Locally

```
git clone https://github.com/your-username/tic-tac-toe.git
cd tic-tac-toe
open index.html
```

## Game Rules

* Played on a 3x3 grid
* Player 1 uses X, Player 2 uses O
* Players take turns
* First to align 3 marks wins
* If all cells are filled, the game is a draw
